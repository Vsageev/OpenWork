# Agent Chat Turn Lifecycle Regression Matrix

Extend this matrix when chat turn, queue, branch, or transcript rendering behavior changes.

| Scenario                                                                                                   | Backend API view coverage                                                                                                                                                                                                                                          | Frontend/browser coverage                                                                                                |
| ---------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------ |
| Send first prompt, stop run, send follow-up: follow-up appears as a new turn                               | `agent-chat-view.contract.test.ts` -> `renders a stopped first prompt followed by a normal follow-up turn`                                                                                                                                                         | `agent-chat-lifecycle.browser.test.ts` -> `shows a stopped first prompt and the follow-up as a new visible turn`         |
| Send first prompt, stop run, edit first prompt: edit appears as explicit replacement, not normal follow-up | `agent-chat-view.contract.test.ts` -> `renders an edit of the first prompt as an explicit replacement, not a follow-up`                                                                                                                                            | `agent-chat-lifecycle.browser.test.ts` -> `shows an edited first prompt as a replacement row instead of a follow-up row` |
| Edit a prior prompt, then send a follow-up on the edited branch                                            | `agent-chat-view.contract.test.ts` -> `renders a follow-up on an edited prior prompt branch`                                                                                                                                                                       | `agent-chat-lifecycle.browser.test.ts` -> `keeps a follow-up on the edited branch after editing a prior prompt`          |
| Queue multiple prompts while a run is active                                                               | `agent-chat-view.contract.test.ts` -> `renders multiple queued prompts behind an active run in turn order`                                                                                                                                                         | `agent-chat-lifecycle.browser.test.ts` -> `renders multiple prompts queued behind an active run as queued rows`          |
| Upload with caption and attachments, then follow-up                                                        | `agent-chat-view.contract.test.ts` -> `renders upload caption attachments followed by a normal follow-up`                                                                                                                                                          | `agent-chat-lifecycle.browser.test.ts` -> `renders upload caption attachments and then the follow-up turn`               |
| Failed/cancelled queue item retry and removal                                                              | `agent-chat-view.contract.test.ts` -> `renders failed and cancelled queue items with retry/removal controls`; mutation sync remains covered by `agent-chat.turn-write-path.test.ts` -> `keeps retry, remove, and reorder queue operations in sync with turn state` | `agent-chat-lifecycle.browser.test.ts` -> `renders failed and cancelled items with retry/removal controls`               |
